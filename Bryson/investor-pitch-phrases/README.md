# 投资者路演英语短语库

## 📋 项目概述
为雅思口语商业场景和投资者路演开发的专用英语短语库，包含8个核心类别、50个关键短语，支持立即用于英语培训和雅思备考。

## 🎯 设计目标
1. **实战导向** - 源自真实投资对话和路演场景
2. **分层学习** - 从基础表达到高级策略
3. **结构化练习** - 短语卡、音频模板、练习脚本
4. **雅思适配** - 结合Part 2和Part 3的考试要求

## 🎨 短语库特点

### 分层难度设计
- **🔵 基础级** (适合雅思5.5-6.0): 简洁清晰的表达
- **🟡 进阶级** (适合雅思6.5-7.0): 商业专业术语
- **🔴 高级级** (适合雅思7.5+): 战略谈判和复杂概念表达

### 学习路径设计
1. **记忆阶段**: 理解和记忆单个短语
2. **应用阶段**: 在情境中练习使用
3. **整合阶段**: 构建完整的路演表达
4. **创作阶段**: 个性化改编和创造

### 雅思考试适配
**Part 2 长独白**:
- 使用短语库构建2分钟路演话题卡
- 商业产品/服务推荐的场景应用

**Part 3 深入讨论**:
- 创业者应具备的品质
- 投资风险评估的思考
- 商业发展趋势预测

## 🚀 快速开始

### 第一天练习计划 (30分钟)
1. **5分钟**: 学习开场白类别的前3个短语
2. **10分钟**: 跟读练习，注意重音和语调
3. **10分钟**: 根据模板创建微型路演开头 (45秒)
4. **5分钟**: 录音并自我评估

### 每周学习循环
- **周一**: 学习新短语 (20分钟)
- **周二**: 跟读练习 (15分钟)  
- **周三**: 模拟对话 (20分钟)
- **周四**: 完整路演 (25分钟)
- **周五**: 回顾测试 (15分钟)

## 📊 预期学习成果

### 4周学习计划
```
第1周: 掌握前3个类别的基础表达
第2周: 能够完成3分钟的简单路演
第3周: 熟练使用商业术语和谈判策略
第4周: 掌握完整投资者对话的流程控制
```

### 雅思分数对应
- **6.0分**: 能够使用库中基础短语完成商业话题
- **6.5分**: 能自然组合短语并有逻辑性地表达
- **7.0分**: 熟练运用高级表达，表现出商业洞察力
- **7.5分+**: 能够个性化调整短语，展示创造力

## 💡 教学建议

### 最佳实践
1. **先理解，后记忆** - 理解短语的使用场景至关重要
2. **小批量学习** - 每次学习3-5个短语，确保掌握
3. **情境化练习** - 模拟真实投资对话场景
4. **录音自评** - 定期录音并与目标音频对比
5. **渐进式难度** - 从基础短语开始，逐步挑战更高难度的表达

### 常见问题解决
**问题**: "我记住了短语，但真正对话时就忘记了"
**方案**: 使用`exercises/role_play_scenarios.md`的场景反复练习，建立肌肉记忆

**问题**: "我的口音会影响表达效果吗?"
**方案**: 优先关注语调模式和重音位置，口音不是主要评分因素

**问题**: "雅思考官不了解我的商业概念怎么办?"
**方案**: 按照`resources/advanced_expressions.md`的解释技巧，用简单语言解释复杂概念
