# Cloudflare Tunnel 快速设置指南 (Bryson语音MVP)

## 📋 **您需要完成的任务清单**

### **任务1: Cloudflare账号注册** ⏰ 3分钟
**步骤:**
1. 打开 [Cloudflare注册页面](https://dash.cloudflare.com/sign-up)
2. 输入邮箱和密码注册
3. **重要**: 选择 **免费计划** (Free Plan)
4. 完成邮箱验证

**完成标志**: 登录到Cloudflare Dashboard

---

### **任务2: 域名选择决策** ⏰ 2分钟
请从以下选项中选择一种:

#### **选项A: 使用现有域名** (推荐)
- **前提**: 您需要控制此域名的DNS设置
- **操作**: 将域名的DNS服务器改为Cloudflare的
- **优点**: 专业，长期稳定
- **示例**: `bryson-voice.yourdomain.com`

#### **选项B: 使用Cloudflare免费子域名**
- **地址**: 类似 `bryson-tts-xxxx.workers.dev`
- **优点**: 立即可用，无需购买域名
- **缺点**: 不便于记忆，不够专业
- **创建方式**: 自动生成，无需设置

#### **选项C: 购买新域名**
- **建议平台**: Namecheap (便宜) 或 腾讯云
- **价格**: 约 ¥15-50 每年
- **域名建议**: `bryson-tts.com` 或 `ielts-voice.com`

## ⚠️ **重要注意事项**

### **网络要求:**
- Cloudflare Tunnel使用标准HTTPS端口 (443)
- 手机网络需要能访问国际CDN
- 企业网络可能会有防火墙限制

### **备用方案:**
如果Cloudflare Tunnel遇到网络问题，我们还有备选方案:
1. **SSH隧道修复** (需配置SSH密钥)
2. **bore-cli工具** (无需认证的轻量级隧道)
3. **pagekite** (Python实现)
