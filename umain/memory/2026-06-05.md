## WorkBuddy深度调研（Daryl授权直接查看本机安装）

### 关键发现
- WorkBuddy = 腾讯CodeBuddy的桌面AI Agent产品
- 底层引擎：Electron + CodeBuddy CLI（fork自Claude Code类架构）
- 默认模型：DeepSeek-V4-Pro（从进程参数确认）
- 本地数据库：SQLite（workbuddy.db）
- 已配置渠道：飞书Bot、微信公众号、微信CrawBot

### 核心架构分析
1. **工具链**：Read/Write/Edit/Bash/WebFetch/WebSearch/Skill/SkillManage/TaskCreate/ImageGen/Agent/TeamCreate/SendMessage + MCP tools
2. **MCP集成**：connector-proxy（HTTP MCP server，本地端口）
   - 内置tools: preview_url, open_result_view, connect_cloud_service, deliver_attachments, workbuddy_cloudstudio_deploy
   - Ardot设计工具（Figma-like）: 20+ ardot_* tools
3. **Skill系统**：与OpenClaw类似的SKILL.md结构，但有更完整的生态（60+官方plugins）
4. **部署系统**：CloudStudio Deploy - 一键将静态站点部署到云端沙箱，生成shareLink
5. **项目隔离**：每个对话创建独立项目目录（~/WorkBuddy/时间戳/）
6. **Artifact系统**：artifact-index追踪每个session的产物（文件、任务等）

### vs OpenClaw对比
WorkBuddy优势：
1. **可视化界面** - 完整的Electron GUI，实时看到Agent工作
2. **产物追踪** - artifact-index自动记录所有生成的文件
3. **一键部署预览** - CloudStudio Deploy生成即时访问URL
4. **技能生态** - 60+官方plugins + 用户自建skills
5. **沙箱隔离** - sandbox-cli为每个任务提供隔离环境
6. **MCP集成** - 通过connector-proxy统一管理工具调用

OpenClaw优势：
1. **多渠道通信** - 飞书/Telegram/Signal/Discord原生支持
2. **持久化Agent** - 24/7运行的后台守护进程
3. **心跳机制** - 主动检查和汇报
4. **跨Agent通信** - sessions_spawn/sessions_send
5. **Cloudflare生态** - Workers/Pages/KV/R2深度集成
