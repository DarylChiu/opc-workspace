# active.md 统一模版 v1.0

> OPC 看板解析器绑定此格式。不按此模版的 active.md → 看板数据不完整。
> 所有 Agent 强制执行，修改后通知 Kitty 验证。

---

## 标题：三个固定标题，一字不能改

```markdown
## 🟢 进行中
## ✅ 已完成
## 🔵 待办
```

- 必须是这三个精确字符串，不含任何后缀或变体
- ❌ `## 🔴 进行中 → M2` ❌ `## 🟢 运维` ❌ `## ⏸️ 暂停/等待`
- 看板解析器**只识别这三个 `##` 标题**，其余 `##` 全部忽略

## 任务条目：必须用 `### `

```markdown
## 🟢 进行中

### 任务名称 — 简短状态描述
- 关键进度
- **下一节点**: xxx
- **阻塞**: xxx（如有）
```

- 一个 `### 任务名` = 看板上的一个任务卡片
- ❌ 不允许用 `- item` 或 `- [ ] item` 代替
- 详细信息用 `- ` 列表放在 `### ` 下方

## 已完成：仅保留最近 7 天

```markdown
## ✅ 已完成

### 已完成任务名 (YYYY-MM-DD)
- 关键产出
```

超过 7 天的已完成任务移入日记归档，保持 active.md 精简。

## 暂停/等待的任务：归入待办

标记 ⏸️ 前缀即可，依然用 `### `：

```markdown
## 🔵 待办

### ⏸️ 等待 Daryl 反馈 — 某方案
- 已出具方案，等待确认

### 待启动任务
- 准备中
```

## 禁止事项

| ❌ 禁止 | ✅ 正确 |
|---------|--------|
| `## ` 标题加后缀或变体 | `## 🟢 进行中` 一字不改 |
| 用 `-` 或 `- [ ]` 代替 `###` | 每个独立任务都用 `### ` |
| 超过 3 个 `##` 标题 | 只保留 `🟢 进行中` `✅ 已完成` `🔵 待办` |
| 已完成堆超过 2 周 | 超过 7 天移入日记 |
