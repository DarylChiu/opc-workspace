## 知识体系结构（5 大模块）

### Section A: Business Organisation, Structure and Governance (20%)

#### A1. Business Organisation and Structure
**核心概念**:
- 企业类型分类（Sole trader / Partnership / Limited company / Public sector）
- 利润导向 vs 非营利组织
- 微观/宏观/全球化商业环境
- PEST 分析（Political / Economic / Social / Technological）
- Porter's Five Forces（五力模型）
- 商业生命周期（Startup → Growth → Maturity → Decline）

**关键问题**:
- 不同企业类型的法律责任区别？
- 非营利组织的 "stakeholder" 定义与营利企业有何不同？
- PEST 与 PESTLE（加 Legal/Environmental）的区别？

---

#### A2. Stakeholders
**核心概念**:
- Stakeholder 定义（Mendelow's Matrix: Power vs Interest）
- Internal stakeholders（员工、管理层、股东）
- External stakeholders（客户、供应商、政府、社区）
- Connected vs External stakeholders
- Stakeholder conflict（利益冲突场景）
- Corporate Social Responsibility (CSR)

**关键问题**:
- Shareholder vs Stakeholder 理论的本质区别？
- Mendelow Matrix 的四个象限策略（Keep satisfied / Keep informed / Minimal effort / Key players）
- CSR 是成本还是投资？（与 Porter's Value Chain 的关系）

---

#### A3. Business Organisational Structure, Constitution and Governance
**核心概念**:
- 组织结构类型（Functional / Divisional / Matrix / Network）
- Span of control（管理幅度）vs Levels of hierarchy
- Centralisation vs Decentralisation
- Corporate governance 原则（Transparency / Accountability / Independence）
- Board of directors（Executive vs Non-executive directors, NEDs）
- Audit committee / Remuneration committee / Nomination committee

**关键问题**:
- Matrix structure 的优缺点？（双重汇报关系的冲突）
- NEDs 的独立性如何保证？（与 Executive directors 的利益冲突）
- Governance vs Management 的边界在哪里？

---

#### A4. Business Organisational Culture, Committees and Meetings
**核心概念**:
- Organisational culture 模型（Handy / Schein / Hofstede）
- Power culture / Role culture / Task culture / Person culture
- Culture change（Lewin's 3-stage model: Unfreeze → Change → Refreeze）
- Committee 类型与职能
- Meeting 流程（Agenda / Minutes / Quorum / Motions / Resolutions）

**关键问题**:
- Hofstede 的文化维度（Power distance / Individualism / Uncertainty avoidance 等）与企业决策的关系？
- Culture 能被"设计"吗？还是只能"演化"？
- Committee 与 日常管理团队 的区别？

---

### Section B: Business Organisational Strategy and Technology (25%)

#### B1. Strategic Management
**核心概念**:
- Strategy 层次（Corporate / Business / Functional / Operational）
- Strategic planning（Mission → Vision → Objectives → Strategy → Tactics）
- SWOT 分析（Strengths / Weaknesses / Opportunities / Threats）
- Ansoff Matrix（Market penetration / Product development / Market development / Diversification）
- Porter's Generic Strategies（Cost leadership / Differentiation / Focus）
- BCG Matrix（Stars / Cash cows / Question marks / Dogs）
- Strategic change（Emergent vs Planned strategy）

**关键问题**:
- Mission vs Vision 的本质区别？（是否 Mission = 现在，Vision = 将来？）
- SWOT 如何转化为实际战略？（SO / WO / ST / WT 策略矩阵）
- Ansoff Matrix 中的 Diversification 为什么风险最高？
- BCG Matrix 的"相对市场份额"如何计算？
- Porter 的 "Stuck in the middle" 是否永远不可行？

---

#### B2. Information Technology
**核心概念**:
- IT 在商业中的角色（Operational / Managerial / Strategic）
- Big Data（Volume / Velocity / Variety / Veracity）
- Cloud computing（SaaS / PaaS / IaaS）
- E-commerce / E-business
- Cybersecurity（Threats: Malware / Phishing / Ransomware / DDoS）
- Data protection（GDPR 原则）
- IT controls（Access controls / Backup / Disaster recovery）

**关键问题**:
- Big Data 的"Veracity"（真实性）如何验证？
- Cloud computing 的成本优势 vs 数据主权风险？
- GDPR 的"被遗忘权"（Right to be forgotten）在实务中如何实现？
- IT 战略与业务战略的对齐（IT-Business alignment）

---

#### B3. Business Intelligence and Data Analytics
**核心概念**:
- Business Intelligence (BI) 定义
- Data analytics 类型（Descriptive / Diagnostic / Predictive / Prescriptive）
- Data mining（Pattern recognition / Classification / Clustering）
- Data visualisation（Dashboard / KPI / Balanced Scorecard）
- Management Information Systems (MIS)
- Decision Support Systems (DSS)

**关键问题**:
- BI vs Data Analytics 的区别？
- Predictive analytics 的准确度如何评估？
- KPI 设计的 SMART 原则（Specific / Measurable / Achievable / Relevant / Time-bound）
- Balanced Scorecard 的四个维度（Financial / Customer / Internal Process / Learning & Growth）如何平衡？

---

#### B4. Project Management
**核心概念**:
- Project 定义（Temporary / Unique / Progressive elaboration）
- Project lifecycle（Initiation → Planning → Execution → Monitoring & Control → Closure）
- Project management constraints（Time / Cost / Scope / Quality）— Triple constraint
- Gantt chart / Critical Path Method (CPM) / PERT
- Risk management（Identify → Assess → Mitigate → Monitor）
- Stakeholder management in projects

**关键问题**:
- Project vs BAU (Business As Usual) 的本质区别？
- Critical Path 如何计算？（Forward pass / Backward pass）
- Triple constraint（时间/成本/范围）冲突时如何取舍？
- Agile vs Waterfall 项目管理方法的适用场景？

---

### Section C: Human Resource Management (20%)

#### C1. Individual and Group Behaviour
**核心概念**:
- Personality 理论（Big Five: OCEAN — Openness / Conscientiousness / Extraversion / Agreeableness / Neuroticism）
- Perception（Selective perception / Halo effect / Stereotyping）
- Attitudes（Cognitive dissonance）
- Group dynamics（Forming → Storming → Norming → Performing → Adjourning）— Tuckman
- Groupthink（风险与预防）
- Social loafing（搭便车效应）

**关键问题**:
- Big Five 与工作绩效的相关性？（哪个维度最重要？）
- Cognitive dissonance（认知失调）如何影响员工行为（如离职倾向）？
- Groupthink 的经典案例（如 Challenger 号灾难）的预防机制？
- 虚拟团队（Virtual team）如何克服 social loafing？

---

#### C2. Recruitment and Selection
**核心概念**:
- Recruitment process（Job analysis → Job description → Person specification → Advertising）
- Internal vs External recruitment
- Selection methods（Interview / Assessment centre / Psychometric tests / References）
- Validity vs Reliability（选拔方法的评估）
- Bias in selection（Confirmation bias / Halo effect / Similarity bias）

**关键问题**:
- Job description vs Person specification 的区别？
- 哪种选拔方法的 validity 最高？（结构化面试 vs 非结构化面试）
- Psychometric tests 的法律风险（如歧视）？
- Internal recruitment 的优势 vs "近亲繁殖"风险？

---

#### C3. Diversity and Inclusion in the Workplace
**核心概念**:
- Diversity 定义（Age / Gender / Ethnicity / Disability / Sexual orientation / Religion）
- Inclusion vs Diversity（包容 vs 多样性）
- Business case for diversity（创新、决策质量、雇主品牌）
- Unconscious bias（隐性偏见）
- Diversity policies（Quota vs Target / Positive action vs Positive discrimination）

**关键问题**:
- Diversity 的 "business case" 是否成立？（证据强度）
- Quota（配额制）是否是一种"反向歧视"？
- Unconscious bias training 是否有效？
- Virtual workplace 对 diversity 的影响（远程工作 vs 传统办公）？

---

#### C4. Training and Development
**核心概念**:
- Training vs Development vs Education
- Training needs analysis（Organisational / Task / Person analysis）
- Training methods（On-the-job / Off-the-job / E-learning / Coaching / Mentoring）
- Training evaluation（Kirkpatrick's 4 levels: Reaction / Learning / Behaviour / Results）
- Continuous Professional Development (CPD)

**关键问题**:
- On-the-job training 的成本优势 vs 质量风险？
- Coaching vs Mentoring 的区别？
- Kirkpatrick Level 4（Results）如何衡量？（ROI of training）
- CPD 对 ACCA 会员的强制要求（多少小时？）

---

#### C5. Performance Appraisal and Review
**核心概念**:
- Performance appraisal 目的（Development / Reward / Promotion / Feedback）
- Appraisal methods（Trait-based / Behaviour-based / Results-based / 360-degree feedback）
- Rating scales（Graphic rating scale / Behaviorally Anchored Rating Scales, BARS）
- Appraisal bias（Leniency / Strictness / Central tendency / Recency effect / Halo effect）
- Performance Improvement Plan (PIP)

**关键问题**:
- 360-degree feedback 的优势 vs "popularity contest" 风险？
- Trait-based appraisal（如"态度""主动性"）是否主观性过强？
- Recency effect（近期效应）如何避免？（持续记录 vs 年度评估）
- PIP 是"改进工具"还是"解雇前奏"？

---

#### C6. Reward Systems
**核心概念**:
- Reward 类型（Financial vs Non-financial / Intrinsic vs Extrinsic）
- Financial rewards（Base pay / Bonus / Commission / Profit-sharing / Share options）
- Non-financial rewards（Recognition / Autonomy / Career development / Work-life balance）
- Total reward concept
- Pay structures（Job evaluation / Market pricing / Pay grades）
- Executive pay（争议与 governance）

**关键问题**:
- Financial rewards 是否总是最有效？（Herzberg 的"Hygiene factors"）
- Bonus 与长期绩效的关系？（短期主义风险）
- Share options 对员工保留率的影响？（Golden handcuffs）
- CEO pay ratio（CEO 与普通员工薪酬倍数）的合理范围？

---

### Section D: Leadership, Management and Supervision (20%)

#### D1. Leadership, Management and Supervision
**核心概念**:
- Leadership vs Management（领导 vs 管理的本质区别）
- Leadership theories:
  - Trait theory（天生 vs 后天）
  - Style theory（Autocratic / Democratic / Laissez-faire）
  - Contingency theory（Fiedler / Situational leadership — Hersey & Blanchard）
  - Transformational vs Transactional leadership
- Management functions（Fayol's POLC: Planning / Organising / Leading / Controlling）
- Supervision（First-line management）

**关键问题**:
- "Leaders do the right thing, managers do things right" — Bennis 的定义是否绝对？
- Trait theory 是否已过时？（现代研究的结论）
- Situational leadership 的四种风格（Telling / Selling / Participating / Delegating）如何判断适用场景？
- Transformational leadership 是否总是优于 Transactional？（是否存在场景依赖）

---

#### D2. Motivation and Empowerment
**核心概念**:
- Motivation theories:
  - Content theories（关注"需求"）— Maslow / Herzberg / Alderfer (ERG) / McClelland
  - Process theories（关注"过程"）— Expectancy theory (Vroom) / Equity theory (Adams) / Goal-setting theory
- Empowerment（授权 vs 放任）
- Job design（Job enrichment / Job enlargement / Job rotation / Flexible working）

**关键问题**:
- Maslow 的需求层次是否普遍适用？（文化差异？）
- Herzberg 的"Motivation factors" vs "Hygiene factors" 如何验证？
- Expectancy theory 的三要素（Expectancy / Instrumentality / Valence）在实务中如何应用？
- Empowerment 的前提条件（能力/信任/资源）？

---

#### D3. Team Dynamics and Team Building
**核心概念**:
- Team vs Group（区别）
- Belbin's Team Roles（9 种角色，如 Plant / Shaper / Coordinator）
- Team effectiveness（Hackman's model: Real team / Compelling direction / Enabling structure / Supportive context / Expert coaching）
- Cross-functional teams / Virtual teams / Self-managed teams
- Team conflict（Task vs Relationship conflict / Conflict resolution styles — Thomas-Kilmann）

**关键问题**:
- Belbin 的 9 种角色是否需要全部齐备？
- Virtual team 的挑战（时区/沟通/信任）如何克服？
- Task conflict 是否有益？（vs Relationship conflict）
- Thomas-Kilmann 的五种冲突解决风格（Competing / Collaborating / Compromising / Avoiding / Accommodating）的适用场景？

---

### Section E: Professional Ethics (15%)

#### E1. Ethical Considerations in Business and Accountancy
**核心概念**:
- Ethics 定义（Moral principles governing behaviour）
- Ethical theories:
  - Deontology（义务论，Kant）— 行为本身的对错
  - Consequentialism / Utilitarianism（后果论，Bentham / Mill）— 结果最大化
  - Virtue ethics（德性论，Aristotle）— 品格导向
- Corporate ethics（Ethical culture / Tone at the top）
- Business ethics issues（Bribery / Corruption / Fraud / Money laundering / Tax avoidance vs Tax evasion）

**关键问题**:
- Deontology vs Utilitarianism 在实际决策中如何选择？（如"电车难题"）
- "Tone at the top" 如何落地？（文化 vs 制度）
- Tax avoidance（合法避税）vs Tax evasion（逃税）的道德边界？
- Whistleblowing（举报）的保护机制？

---

#### E2. Codes of Conduct, Ethics and Law
**核心概念**:
- ACCA Code of Ethics and Conduct（五大原则）:
  1. Integrity（诚信）
  2. Objectivity（客观）
  3. Professional competence and due care（专业能力与勤勉）
  4. Confidentiality（保密）
  5. Professional behaviour（专业行为）
- Threats to fundamental principles（Self-interest / Self-review / Advocacy / Familiarity / Intimidation）
- Safeguards（Organisational / Professional / Legal）
- Law vs Ethics（Legal ≠ Ethical）

**关键问题**:
- 五大原则之间冲突时如何取舍？（如 Confidentiality vs Public interest）
- Self-review threat 如何避免？（审计师审计自己做的账）
- Familiarity threat 的典型场景？（长期合作关系）
- 合法但不道德的行为？（如零工经济中的"合同工"身份规避）

---

#### E3. Ethical Conflict
**核心概念**:
- Ethical dilemma（道德困境）
- Conflict resolution（识别 → 评估 → 行动）
- Reporting mechanisms（Internal vs External / Whistleblowing hotline）
- Ethical decision-making frameworks（AAA model: Analyse / Act / Account）
- Professional scepticism

**关键问题**:
- 典型道德困境场景？（如老板要求做假账 / 同事行贿 / 发现客户欺诈）
- Whistleblowing 的保护法律（如 UK Public Interest Disclosure Act）
- Professional scepticism vs Cynicism（怀疑 vs 愤世嫉俗）的边界？
- ACCA 会员违反 Code of Ethics 的后果？（纪律处分 / 撤销会员资格）
