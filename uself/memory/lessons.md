## 跨域财税方法论

> 来源: FutureTextile × Wellname 股东借款利息案例 (Balance 攻防对抗, 2026-07-01)
> 去重结果: 6 条均为财税法律领域新方法论，与现有工程/流程类方法论无重叠

### 1. 法律概念选择决定税务结果
- **核心**: 措辞创造或消灭税基。waiver（利息存在过→被免除→应税事件）vs condition precedent（条件未成就→利息从未存在→零税务事件）
- **适用范围**: 任何跨境关联交易的合同条款设计
- **反面教材**: 双叙事自毁——"4%是市场利率"(TP防御) + "实质是注资"(CIT防御) 互斥，不能同时主张

### 2. 消灭红旗 = 消灭审计
- **核心**: 越南税务体系是自报自缴 + 红旗触发审计。让申报表无任何可质疑变动，等于消灭了稽查入口
- **适用范围**: 越南/类似自报自缴体系下的税务合规策略
- **边界**: 不适用于以税务稽查为常态的司法管辖区

### 3. FCT 收付实现制
- **核心**: 外国承包商税(FCT)纳税义务在实际支付时产生，不是计提时。利息从未支付 → FCT 风险天然为零
- **适用范围**: 越南 FCT 相关交易（利息/特许权使用费/服务费）
- **关键区分**: CIT 和 FCT 的纳税时点不同，不要混用

### 4. 行政时点法律效力分层
- **核心**: 协议 < IRC(投资登记证) < ERC(企业登记证)。越靠近工商登记端，外部证据效力越强
- **适用范围**: 任何依赖行政审批时点作为会计分录依据的场景
- **判断标准**: "这个文件证明的是'可以做某事'还是'某事已完成'？"

### 5. 银行汇款附言策略
- **核心**: 第三方独立证据（银行记录）不可控——银行不会按你的需求写附言。需要用可转换工具条款 + 董事会决议桥接
- **适用范围**: 任何需要第三方独立证据但无法控制证据内容的场景
- **替代方案**: 不依赖单点证据，用合同条款 + 公司决议 + 外部登记文件形成证据链闭环

### 6. 三方案对比分析框架
- **核心**: 豁免 / 无息 / 可转换工具，按 TP风险、CIT敞口、被挑战后推定利率 三个维度量化对比
- **适用范围**: 任何多方案决策场景（尤其涉及跨境税务）
- **关键**: 被挑战后的最坏情况必须量化——不能只说"方案B风险更高"，要算"风险高多少倍"
