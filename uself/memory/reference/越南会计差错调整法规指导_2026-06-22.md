## 二、会计处理：VAS 29 差错更正框架

### 2.1 差错的分类

| 类型 | 定义 | 例子 |
|------|------|------|
| **重大差错 (Material)** | 单独或汇总可能影响报表使用者经济决策 | 少计收入、错计折旧、漏记大额费用 |
| **非重大差错 (Immaterial)** | 金额/性质不显著影响报表使用者决策 | 小额分类错误、不影响净利润的调整 |

### 2.2 差错发现时点的处理原则（核心！）

```
                   差错发生时间
                        │
        ┌───────────────┼───────────────┐
        │               │               │
   当期发生          以前期间发生      以前期间发生
   (2026年内)       (2025年)         (2024年及以前)
        │               │               │
        ▼               ▼               ▼
  在当期损益中       区分两种场景      区分两种场景
  直接调整           ↓               ↓
                ①报表批准前：     ①报表批准前：
                调整当期报表       调整对应期间报表
                                  再重新出具
                ②报表批准后：     
                ┌──────┬──────┐   ②报表批准后（已无法修改原报表）：
                │ 重大 │ 非重大│   
                │      │      │   ┌────────┬────────┐
                ▼      ▼      │   │ 重大   │ 非重大 │
           追溯重述  当期调整  │   ▼        ▼        │
           ↓                  │ 追溯重述   当期调整   │
    调整期初留存收益          │ ↓                  │
    (4211) + 重述可比数据     │ 调整4211期初留存    │
                              │ + 重述可比数据      │
                              └────────────────────┘
```

**简单记忆法**：
- **非重大差错** → 在哪发现就在哪调（走当期损益）
- **重大前期差错（报表已出）** → 追溯重述，调整期初留存收益(4211)，不走当期P&L
- **重大当期差错** → 当期调整

### 2.3 VAS 29 vs IFRS/IAS 8 关键差异

| 项目 | VAS 29 | IAS 8 |
|------|--------|-------|
| 追溯调整的不可行性处理 | 表述较简略 | 详细规定 |
| 披露要求 | 较简略 | 非常详细（含对EPS的影响） |
| 税务影响处理 | 引用VAS 17（但VAS 17未完整涵盖递延税） | 完整递延税处理 |

**⚠️ 实务注意**：VAS 29执行时，税务影响参照VAS 17（所得税），但VAS 17对递延所得税的处理不完整。实务中**调整前期差错时不要忽略对企业所得税的影响**。

## 三、具体会计分录模板（Circular 200科目体系）

### 3.1 非重大差错 — 当期发现，当期调整

**场景A：少计费用**（如漏记折旧）
```
Dr  相关费用科目 (641/642/627)    XXX
    Cr  累计折旧 (214)                   XXX
```
→ 影响当期利润表，直接减少当期净利润

**场景B：多计收入**（如提前确认未实现收入）
```
Dr  营业收入 (511)                XXX
    Cr  应收账款 (131)                    XXX
```
→ 减少当期收入，影响当期利润

### 3.2 重大前期差错 — 追溯重述（报表已批准后才发现）

**场景C：上年少计提折旧 VND 50,000,000（CIT税率20%）**

会计分录：
```
① 更正错误本身：
Dr  4211 - 留存收益-以前年度      50,000,000
    Cr  累计折旧 (214)                  50,000,000

② 税务影响（少计费用=多交税，应退税/抵减）：
Dr  应收退税/预付税款 (3334)      10,000,000  (=50M × 20%)
    Cr  4211 - 留存收益-以前年度         10,000,000
```
> 净影响：4211减少 40,000,000（税后）

**场景D：上年多计收入 VND 100,000,000**

会计分录：
```
① 更正错误本身：
Dr  4211 - 留存收益-以前年度     100,000,000
    Cr  应收账款 (131)                 100,000,000

② 税务影响（多计收入=多交税）：
Dr  应收退税/预付税款 (3334)      20,000,000
    Cr  4211 - 留存收益-以前年度         20,000,000
```
> 净影响：4211减少 80,000,000（税后）

**场景E：上年少计收入 VND 80,000,000**

会计分录：
```
① 更正错误本身：
Dr  应收账款 (131)                80,000,000
    Cr  4211 - 留存收益-以前年度          80,000,000

② 税务影响（少计收入=少交税=要补税）：
Dr  4211 - 留存收益-以前年度      16,000,000
    Cr  应交税费-企业所得税 (3334)         16,000,000
```
> 净影响：4211增加 64,000,000（税后）

### 3.3 关键科目说明

| 科目代码 | 名称 | 用途 |
|----------|------|------|
| **4211** | Lợi nhuận sau thuế chưa phân phối năm trước（以前年度未分配利润） | 前期差错更正的落脚点 |
| **4212** | Lợi nhuận sau thuế chưa phân phối năm nay（本年度未分配利润） | 当年利润分配 |
| **3334** | Thuế TNDN（企业所得税） | CIT应交/应退 |
| **8211** | Chi phí thuế TNDN hiện hành（当期CIT费用） | 当期所得税费用 |

### 3.4 资产负债表期初数调整

除了分录，重大前期差错的追溯重述还需要：
1. **调整本期资产负债表期初数** — 相关资产负债科目按更正后金额列示
2. **重述可比期间数据** — 利润表"上年同期"列按更正后数据列示
3. **附注披露** — 说明差错性质、更正金额、对各科目影响

## 四、税务处理 — 核心风险点 ⚠️

### 4.1 补充申报机制（Kê khai bổ sung）

依据：Law on Tax Administration 2019 + Decree 126/2020/NĐ-CP

**可以主动补充申报的情形**：
- 纳税人在税务机关检查/审计前**主动发现**申报错误
- 在原申报期内提交补充申报表（Tờ khai bổ sung）

**补充申报的税务后果因场景而异**：

| 场景 | 会计利润影响 | 税务处理 | 罚则风险 |
|------|-------------|----------|----------|
| **少计费用→当期利润虚高** | 调减留存收益 | 可申请退税或抵扣以后期间CIT | ✅ 主动补报可免罚款 |
| **多计费用→当期利润虚低** | 调增留存收益 | **补缴CIT差额+滞纳金** | ⚠️ 滞纳金0.03%/天 |
| **少计收入→当期利润虚低** | 调增留存收益 | **补缴CIT+滞纳金** | ⚠️ 可能被认定逃税 |
| **多计收入→当期利润虚高** | 调减留存收益 | 申请退税或抵扣 | ✅ 但需证明非故意 |

### 4.2 滞纳金计算（关键！）

- **利率**：0.03%/天（≈10.95%/年）
- **起算日**：原纳税申报截止日次日
- **截止日**：实际补缴日

示例：少申报CIT 100,000,000 VND，逾期365天
→ 滞纳金 = 100M × 0.03% × 365 = **10,950,000 VND**

### 4.3 罚款风险

| 违规类型 | 罚款比例 | 法律依据 |
|----------|----------|----------|
| 主动补报（税务机关未发现） | 可免罚款，仅补税+滞纳金 | Decree 125/2020 |
| 税务机关检查发现少报 | 20%少报税款 | Decree 125/2020 |
| 逃税行为认定 | 1-3倍逃税金额 | Law on Tax Administration |
| 使用非法发票 | 加重处罚 | Decree 125 |

### 4.4 时效

- **税务追征时效**：10年（Law 38/2019/QH14）
- **退税申请时效**：一般5年

## 六、给Daryl的行动建议

### 立即行动
1. **盘点错误清单** — 让下属按科目列出所有发现的错误，标注：科目、金额、发生期间、错误类型（少计/多计/分类错误）
2. **评估重大性** — 用定量（占总资产/净利润的%）+定性（是否影响盈亏方向）判断
3. **分两路处理**：
   - **会计口径**：按VAS 29做分录调整
   - **税务口径**：评估CIT影响，决定是否补充申报

### 注意点（越南特有）
- ⚠️ **越南税务机关对「少计收入」极其敏感** — 哪怕金额不大，主观补报也可能触发税务稽查
- ⚠️ **建议先咨询本地税务顾问**再提交补充申报，尤其是涉及少报收入的情形
- ⚠️ **Circular 99/2025即将生效**（2026.1.1），新科目体系、IFRS机制都会引入，如果差错涉及2026年以后的期间，需要考虑新规则
- 💡 **当前仍在Circular 200体系下**（2026年财政年度之前），按200号的科目体系处理
