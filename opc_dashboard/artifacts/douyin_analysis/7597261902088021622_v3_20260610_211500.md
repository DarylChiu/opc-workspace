## 🎭 导演风格分析

### 1. 纪实主义美学

导演选择**零BGM + 全自然光 + 固定镜头**的极简配置，这是有意为之的美学策略：

- **为什么不用BGM**: BGM会提前告诉观众该有什么情绪。没有BGM时，观众必须自己判断——这恰恰呼应了全片的核心主题：「你以为你看到了真相，但真相往往不是你想的那样」
- **为什么用自然光**: 街头实景拍摄，不做灯光布置 → 强化「这就是真实发生的」纪实幻觉
- **为什么固定镜头为主**: 不提供「导演视角」的引导，让观众如路人般旁观

### 2. 道具驱动叙事

Land Cruiser不是背景道具，而是**叙事引擎**：
- 开场车头特写 → 建立交易对象
- 中段车身作为人物关系的视觉隔离 → 卖家在车左，买家在车右
- 结尾Land Cruiser依然在场 → 无声提醒观众：这一切的起点只是一辆二手车

手机作为**第二道具**的叙事功能：

- 98s低头操作手机 → 回避目光接触
- 手机的暧昧性 → 是便衣证据？还是交易工具？观众和买家一起猜测

### 3. 反打镜头的叙事力量

132s的反打镜头是导演风格的标志性手法：

- **导演不说**: 这是导演最核心的风格——**不告诉你结论，让你自己看**
