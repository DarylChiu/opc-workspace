# ACCA Knowledge Network

Daryl 的个人知识网络 — Obsidian Vault

## 使用方式

1. 用 Obsidian 打开此文件夹作为一个 Vault
2. 入口页面：`00-Overview/Home.md`
3. 按 `Ctrl/Cmd + E` 切换编辑/阅读模式
4. 按 `Ctrl/Cmd + G` 打开图谱视图

## 结构

```
ACCA-Knowledge-Network/
├── 00-Overview/          # 总览仪表盘
├── 01-Financial/         # 财务域 → ACCA → IFRS/VAS
├── 02-Finance/           # 金融域
├── 03-AI-Tech/           # AI 技术域
├── 04-Music/             # 音乐域

```
